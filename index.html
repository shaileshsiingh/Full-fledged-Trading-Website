Frequently asked questions

Live Sessions → Program Selection → Post-Enrolment →

Are the sessions live or recorded?

All the sessions are conducted live via Zoom.

What is the language for the sessions?

The sessions are conducted in both- English and Hindi. Every concept is explained in both English and Hindi Languages.

What if I miss a session?

A repeat recorded session is scheduled in case you end up missing any of the live ZOOM sessions. Try not to skip any of the live sessions.

Can I ask doubts during the live sessions?

The sessions are completely interactive and you’ll be given ample amount of time to ask all of your queries after every concept. Doubts can also be asked via WhatsApp after the sessions.

Which program is best for beginners?

Technical Analysis program and Comprehensive Learning Program are both suitable for beginners. You can go with the Technical Analysis program if you wish to learn only equity and stock analysis. On the other hand, if you wish to learn both - Equity Stock Analysis and F&O, go with the comprehensive program. It’s an All-in-one program.

Are there any pre-requisites for the program?

There are no pre-requisites for the Technical Analysis and comprehensive program as they are both beginners friendly. For future and Options learning program, basic knowledge of price action is expected including support, resistance, trend-line and chart pattern.

How does WhatsApp assistance Work?

You’ll be in direct contact with the Mentor via WhatsApp communication for redressing all of your doubts and queries.

Is any reading material provided with the programs?

Reading material is provided for all the necessary areas.

Will I get any assistance after the sessions are over?

There is post-program assistance included in all the learning programs as mentioned in the curriculum. You are free to contact the mentor after the aforesaid duration as well. Don’t worry! :)
